# Sales-Insight-Data-Analysy


## Project Overview

This project presents an interactive Power BI dashboard developed to analyze sales performance, customer behavior, and key business metrics. The dashboard enables stakeholders to gain actionable insights and supports data-driven decision-making through clear visualizations and dynamic reporting.

---

## Objectives

* Evaluate overall sales performance across different time periods
* Identify top-performing products and categories
* Analyze customer segments and purchasing behavior
* Monitor key performance indicators (KPIs)
* Provide an interactive interface for data exploration

---

## Dataset Description

The dataset used in this project includes the following components:

* Sales data: revenue, profit, and quantity sold
* Customer data: customer segments and geographic distribution
* Product data: categories and sub-categories
* Order data: order dates, regions, and shipping details

---

## Tools and Technologies
### Tools: Power BI | MySQL | DAX | Power Query
* Power BI Desktop
* Power Query for data cleaning and transformation
* DAX (Data Analysis Expressions) for data modeling and calculations
* Data visualization using charts, tables, KPIs, and slicers

---

## Dashboard Features

### Key Features
- MySQL database → Power Query ETL → Star Schema data model
- DAX measures: Revenue, Profit Margin, YoY Growth
- Multi-page dashboard with drill-through & dynamic slicers
- Data quality handling: currency normalization, deduplication
